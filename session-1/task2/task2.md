
# Simple Point data structure

## Task

Complete the source code and header files
- Define the structure: Point(x,y), using typedef
- Complete the distance function and print the results

## Other functions

Write functions for the following:
- `Point p = reflect( Point q )` - return a point reflected in the x-axis, ie. (x,y)-> (x,-y)
- `Point p = shift( Point q, Point dq )` - return a point that is the sum of q and dq

Test your functions with simple data.

Remember to modify both source file and header file when you add a new function.
